# Criando uma Lab Quest

Este tutorial mostra como gerar uma nova quest baseada em um lab AWS.

## Video

[![Watch the tutorial](https://img.youtube.com/vi/KU7fzb0Z0h4/0.jpg)](https://youtu.be/KU7fzb0Z0h4)

## Passos

1. Acesse a pagina **Quest**
2. Escolha um tema AWS
3. Clique em **Generate Quest**
4. Complete as tasks
5. Ganhe XP e badges

## Resultado

Ao finalizar:

- XP acumulado
- Historico salvo
- Badge desbloqueada
